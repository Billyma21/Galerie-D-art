# Galerie d'Art Contemporain — Musée Digital

Un musée digital minimaliste, immersif et élégant pour une artiste peintre contemporaine. Ce projet est conçu pour offrir une expérience visuelle raffinée, mettant en valeur les œuvres d'art à travers une interface épurée et des animations fluides.

## ✨ Fonctionnalités

- **Galerie Immersive** : Présentation des œuvres avec filtres par catégorie et détails complets.
- **Expositions** : Gestion des événements passés et à venir avec localisation et dates.
- **Panneau d'Administration Premium** :
  - Gestion complète des œuvres (CRUD).
  - Gestion des expositions.
  - Personnalisation de l'identité visuelle (titres, textes, images hero, couleurs d'accent).
  - Système de messagerie intégré pour les demandes de contact.
  - Statistiques de visite et de popularité des œuvres.
- **Design Réactif** : Optimisé pour tous les écrans (mobile, tablette, desktop).
- **Animations Fluides** : Utilisation de Framer Motion pour une navigation élégante.

## 🛠️ Stack Technique

- **Frontend** : React 19, TypeScript, Vite
- **Styling** : Tailwind CSS 4
- **Animations** : Framer Motion
- **Icons** : Lucide React
- **Routing** : React Router 7
- **Persistence** : LocalStorage (pour la démonstration)

## 🚀 Installation Locale

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/votre-utilisateur/votre-depot.git
   cd votre-depot
   ```

2. Installez les dépendances :
   ```bash
   npm install
   ```

3. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```

4. Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## 📦 Déploiement sur Vercel

Ce projet est prêt pour un déploiement rapide sur **Vercel**.

1. Connectez votre dépôt GitHub à Vercel.
2. Vercel détectera automatiquement les paramètres Vite.
3. Assurez-vous que la commande de build est `npm run build` et le répertoire de sortie est `dist`.
4. Cliquez sur **Deploy**.
